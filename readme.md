# ARGO Ocean Data Conversational Explorer

An interactive data exploration platform for ARGO oceanographic float data that enables users to query, discover, and visualize ocean measurements using natural language.

The system combines semantic search, vector embeddings, metadata retrieval, and interactive visualizations to make scientific ocean data more accessible and easier to explore.

## Features

* Natural language querying of ARGO float data
* Semantic search using vector embeddings
* ChromaDB-based metadata retrieval
* Dynamic PostgreSQL data access
* Interactive ocean profile visualizations
* Temperature and salinity exploration
* Geographic float discovery
* Conversational scientific data analysis

## Technologies Used

* Python
* Streamlit
* PostgreSQL
* ChromaDB
* LangChain
* OpenAI Embeddings
* Plotly

## Workflow

1. User submits a natural language query
2. Semantic search retrieves relevant metadata
3. Query parameters are extracted automatically
4. Relevant ARGO data is fetched from PostgreSQL
5. Interactive visualizations are generated
6. Users explore ocean profiles and float measurements

## Use Cases

* Oceanographic research
* Scientific data exploration
* ARGO float analysis
* Interactive environmental data visualization
* Natural language access to scientific datasets
